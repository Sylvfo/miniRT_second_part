les parametres

mir: represente l'effet mirroir sur l'object
	sa valeur varie de 0 a 1

pat: represente les patterns sur l'object
	la partie 1 represente le type de motifs il z en a 4
	donc compris entre 0 et 3
	la partie 2 represente juste le nombre de division
	si je puis dire par exemple pour 2 c'est un damier.

	1-damier (checkerboard)
	2-rayures(stripes)
	3-degrades
	4-rond(ring)
	5-morifs proceduraux base sur les fonctions comme le bruit de perlin

ref: represente l'indice de refraction compris entre 1 et 3

tran: la transparence compris entre 0 et 1

txt: represente la texture, mettre le chemin de l'image et d'extension .xpm
